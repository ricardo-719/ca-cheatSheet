In this assignment I am supposed to create a cheatsheet for a particular coding feature I have chosen to cover the topic of creating an awesome button using Tailwind CSS

For the implementation of this assignment I have to use a table in HTML. In this table I will cover a step by step instructive to create an amazing button.

Tailwind Button Instructions

Step Description Notes & References
1 Link Tailwin to the HTML <script src="https://cdn.tailwindcss.com"></script>
2 Create a button container div and set to: absolute with left top and translate x and y as needed
3 Inside create another div containing an anchor for the button: p-4 py-24 px-40 border-4 text-slate-900 border-slate-900
4 Add additional format to that div for hover: hover:text-white hover:after:translate-x-full
5 Include for formating using after: after:absolute after:bg-slate-900 after:right-full after:top-0 after:w-full after:h-full
6 Hide the overflow and add smoothness to the transition: after:duration-300 overflow-hidden
7 Give the Button a tag like this: <h3 class="relative z-20 text-center">Sign as Anonymous</h3>
